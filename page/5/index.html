bower

By LGF

发表于 2017-07-20

前面我们介绍了sass的安装,编译,语法,那么我们现在来看下bower，bower是一个专门下载前端开发中的一些文件的,对我们的开发非常的有用,他更npm是一样的

bower的安装

  • 1.首先你要下载node和git
  • 2.直接window+r调出命令框，然后输入cmd
  • 3.输入：npm install bower -g然后回车
  • 4.前面如果不出意外的话，会安装成功，接着在命令框中输入：bower --version他可以检测bower的版本
  • 5.输入：bower install xxx --save,注意xxx是你要下载的文件的名字，比如我下载的是animate.css，那么xxx就要改成animate.css，然后按下回车马上就行下载，我下载好的文件路径在C:\Users\john\bower_components这里面，–save表示的是联他的配置文件也下载
  • 6.如果你想删除这个下载的文件，就得输入这个命令：bower uninstall xxx,这样我下载的文件就会没有
  • 7.如果你想查看你想要下的文件的一些信息，可以输入这个命令：bower info xxx
  • 8.还有一个很重要的功能，就是包的查找，比如我想要安装vue的某个插件，但是记不住名字了，就可以直接在命令行输入：bower search vue
  • 9.如果你不想下载的东西在c盘下，就可以切换命令行,在你想要下载到那个指定的文件夹里面，shift+鼠标右键会出现一个是否在当前文件夹打开,点击它，这样就切换路径了,此时我下载的东西在d盘中的bower_download_plugin这个文件夹下
  • 10.如果想要更改下载框架的版本可以输入：bower init注意前提是你必须下载了这个框架，然后更具需求填写信息,如果前面的填完后，会自动生成bower.json这个文件,修改这个文件里面的dependencies这个信息，将版本改成你想要的版本，然后保存，接着在命令板中输入bower update这个命令就行更新你修改的内容，然后他会自动下载你规定的版本，然后覆盖以前的版本
  • 11.其实前面的10步更改太麻烦了，可以这样写bower install jquery#1.9.0在后面加上#然后加上你想要的版本号
