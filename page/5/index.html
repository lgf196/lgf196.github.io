vue

By LGF

发表于 2017-07-25

我们今天来聊下vue,总所周知vue是一位中国的人开发的,并且现在的vue是前端开发中可以说是相当的重要的,基本上可以做PC和移动端，不过vue在移动端用的特别的多,并且vue是中国的使用远远要高于react，angular，那么今天我就来介绍下他的运行环境

vue的安装

  • 1.要使用vue来开发，首先要有环境，这个环境要借助于node，所以要先安装node，借助于node里面的npm来安装需要的依赖等等。但是如果在cmd中直接使用npm来安装的一些工具的话会比较慢，所以我们使用淘宝的npm镜像：在命令板中输入:npm install -g cnpm --registry=https://registry.npm.taobao.org,即可安装npm镜像，以后再用到npm的地方直接用cnpm来代替就好了
  • 2.输入cnpm --version来检测是否安装了cnpm
  • 3.输入cnpm install vue-cli -g来安装vue的全局vue-cli脚手架
  • 4.输入vue --version来检测是否安装成功
  • 5.安装完脚手架以后开始，开始创建一个新项目，在命令板中输入vue init webpack vuedemo(最后这个是我创建的项目名),此时一个新的项目文件夹就创建完成，这个项目文件是在使用了脚手架vue-cli的前提下创建的

  • 6.因为各个模板间都是相互依赖的，所以要安装依赖，在命令行输入cnpm install ,你会发现在已经创建的项目结构里面会多出一个node_modules的文件夹，里面就是刚才安装的所有依赖。注意:这个必须是在我下载的那个项目文件里面打开命令板，是我最开始的那个父文件夹
  • 7.准备工作做好以后，测试一下项目里面默认的app.vue模块能否跑起来，这是需要先安装一下服务器环境，在命令行中输入 cnpm run dev 回车即可 ，然后会看到localhost:8080，在游览器的地址栏中输入就可以看到项目的效果，如果下载的是最新版的vue的话,他会自动打开游览器的
